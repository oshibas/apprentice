# ポート番号について説明できる

## 1. ポート番号とは

>ポート番号とは何か、何のためにあるものか、プログラミング初心者にわかるように説明してください。

コンピューター上で動作するプログラムを識別するための番号です。IPアドレスがコンピューターの住所のようなものであるのに対し、ポート番号は部屋番号のようなものです。ポート番号は0から65535までの範囲で指定することができます。

## 2. 代表的なポート番号

>代表的なポート番号はウェルノウンポートと呼ばれています。ウェルノウンポートをいくつか挙げてください。また、それぞれのポート番号が何のために使われているか説明してください。

| ポート番号 | 用途   |
| :--------- | :----- |
| 20         | FTP    |
| 21         | FTP    |
| 22         | SSH    |
| 23         | Telnet |
| 25         | SMTP   |
| 53         | DNS    |
| 80         | HTTP   |
| 110        | POP3   |
| 143        | IMAP   |
| 443        | HTTPS  |

## 3. HTTP/HTTPS 通信

>ブラウザでウェブページを開く際に、ポート番号を指定することができます。

>`https://www.google.com/` をブラウザで開く際に `https://www.google.com:443` を指定しても同じページが開かれますが、`https://www.google.com:22` とするとページが開かれません。その理由を説明してください。

443 番ポートが HTTPS のデフォルトのポート番号であるためです。一方、22 番ポートは SSH のデフォルトのポート番号であり、HTTPS の通信には使用されません。

>また、ブラザでウェブページを開く際に通常はポート番号を指定しませんが、その理由も説明してください。

HTTP の場合はデフォルトで 80 番ポート、HTTPS の場合はデフォルトで 443 番ポートが使用されるためです。

## 4. データベースへの接続

>データベースに接続する際に、ポート番号を指定しています。ポート番号何番を指定しているか確認してください。

データベースごとにデフォルトのポート番号が決まっており、MySQL の場合は 3306 番ポート、PostgreSQL の場合は 5432 番ポートなどが使用されます。

## 5. ポート番号の確認

>今自分が使用しているパソコンで使用しているポート番号とそのポート番号を動かしているプログラムを調べてください。

Mac の場合: ターミナルで lsof -i コマンドを実行します。

```bash
lsof -i
COMMAND     PID         USER   FD   TYPE             DEVICE SIZE/OFF NODE NAME
loginwind   381 ooshibasaori    6u  IPv4 0x733c5ceda3a25bc5      0t0  UDP *:*
...
```
